# Decision Protocol

L0 (default): just do it.
L1: hard-to-reverse decisions (economics, architecture, pricing). Pause, reason carefully inline. No agents needed — just slower thinking.

Escalate because the decision is costly and sticky, not because it feels important.

## Agents

Agents only when the task requires knowledge outside the codebase and conversation.
Everything else: direct tools (Read, Glob, Grep).
